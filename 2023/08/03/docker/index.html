docker 的好处

配置环境是写代码过程中的一个痛点，比如自己写的应用在别人的电脑上或者其他环境下无法运行。

虚拟化是上述问题的一种解决方案，但是虚拟机需要完整的虚拟出一套系统和硬件等，太大了。而应用只需要软件运行环境即可。 docker 是一种轻量化、可移植、可复用、隔离应用和灵活的虚拟化技术。
